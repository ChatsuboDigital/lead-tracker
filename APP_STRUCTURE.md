## User Journey

### First Time User

```
1. Open app â†’ See Upload page
2. Drag CSV file
3. See preview and auto-detection
4. Click "Download Clean CSV"
5. Get clean file + database updated
6. Click "Master Database" tab
7. See all leads in database
8. Search for specific lead
9. Export filtered results
```

### Regular User

```
Morning:
1. Upload new lead list
2. Download clean CSV
3. Import to email tool

Afternoon:
1. Check Master Database
2. Search for specific lead
3. See campaign history
4. Export campaign leads

Weekly:
1. Export all leads
2. Backup database
3. Clean up test leads
```
